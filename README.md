# ⚔️ Solo Leveling: Brendon Edition

一個 RPG 風格的自我提升追蹤系統，讓你像遊戲角色一樣升級自己的生活。

## ✨ 核心功能

### 📊 五大屬性系統
- **STR (體力)** - 運動鍛鍊追蹤（慢跑、重訓、HIIT）
- **INT (智力)** - 能力學習（閱讀、語言、課程）
- **MP (魔力/靈性)** - 靈性探索（讀經、禱告、日記）
- **CRT (創造力)** - 藝術表達（練琴、畫畫）
- **GOLD (財力)** - 事業成長（收入追蹤、關鍵行動）

### 🎮 遊戲化追蹤
- **雷達圖** - 視覺化展示五大維度發展
- **HP 能量條** - 生活作息與飲食健康綜合評分
- **階梯式獎勵** - OK / Great / Best 三級成就系統
- **飲水提醒** - 未達 2400cc 永久置頂提醒

### 💗 特殊系統
- **共鳴 (Resonance)** - 慶祝按鈕配有愛心動畫特效
- **酒精審計** - 非遊戲化的理性記錄區域
- **Google Sheets 整合** - 所有數據自動同步到你的 Google 試算表

## 🚀 快速開始

### 1. 安裝依賴
```bash
npm install
```

### 2. 啟動開發伺服器
```bash
npm run dev
```

### 3. 設置 Google Sheets
首次使用時，應用會引導你：
1. 創建一個新的 Google Sheet
2. 設置權限為「知道連結的任何人都可以編輯」
3. 複製 URL 並貼到應用中

應用會自動在你的 Sheet 中創建所需的欄位結構。

## 📱 使用指南

### 每日任務流程

#### 1. HP (生命) - 最優先
- **飲水追蹤** - 使用 +200, +300, +500 按鈕記錄
  - ⚠️ 未達 2400cc 會持續置頂提醒
- **起床時間** - 05:00 (Best) / 05:30 (Great) / 06:00 (OK)
- **就寢時間** - 21:00 (Best) / 21:30 (Great) / 22:00 (OK)
- **飲食營養** - 一餐自炊 (OK) / 兩餐自炊 (Great) / 兩餐+晚餐禁食 (Best)

#### 2. STR (體力)
勾選今日完成的運動項目：
- 🏃 慢跑
- 🏋️ 重訓
- ⚡ HIIT
完成 1 項=OK，2 項=Great，3 項=Best

#### 3. INT (智力)
- 📚 閱讀 15 分鐘
- 🇮🇹 義大利文 5 分鐘
- 🎓 線上課程 15 分鐘

#### 4. MP (魔力/靈性)
- 📖 讀經
- 🙏 禱告
- ✍️ 靈性日記

#### 5. CRT (創造力)
- 🎹 練琴 10 分鐘
- 🖌️ 畫畫 10 分鐘

#### 6. GOLD (財力)
- 💵 記錄今日收入
- 📈 完成 3 項事業關鍵行動

#### 7. RSN (共鳴/感恩)
- ❤️ 點擊慶祝按鈕（會有愛心特效）
- 📝 寫下感恩筆記

#### 8. 酒精紀錄（如適用）
- 記錄飲酒理由
- 記錄當下心境和身體感受
- 無獎勵機制，純粹自我覺察

### 自動重置
每天凌晨 4:00 自動重置勾選狀態，但保留歷史數據。

## 🛠️ 技術棧

- **React 19** - UI 框架
- **Vite 7** - 快速建構工具
- **Tailwind CSS** - 深色 RPG 風格樣式
- **Recharts** - 雷達圖視覺化
- **Google Sheets API** - 數據持久化
- **localStorage** - 本地數據備份

## 📦 建構與部署

### 建構生產版本
```bash
npm run build
```

### 部署到 Render
專案包含 `render.yaml` 配置，可直接部署：
1. 推送代碼到 GitHub
2. 在 Render 連接你的倉庫
3. 自動讀取配置並部署

詳細步驟請參考 `DEPLOY.md`

## 🎯 設計理念

### 遊戲化但不娛樂化
- 使用 RPG 元素增加動力
- 視覺反饋強化正面習慣
- 酒精紀錄區刻意無遊戲化設計，促進理性反思

### 全面但不繁瑣
- 覆蓋生活五大維度
- 快速操作，不打斷日常流程
- 自動同步，無需手動管理

### 數據驅動自我認識
- 長期追蹤看見成長軌跡
- Google Sheets 支持深度分析
- 覺察模式促進行為改變
