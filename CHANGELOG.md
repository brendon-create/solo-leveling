# 更新日誌

所有重要的專案變更都會記錄在此文件中。

## [2.0.0] - 2026-02-16

### 🎮 重大更新：完全重構為 RPG 風格自我提升系統

這是一個完全重新設計的版本，從簡單的 100 天追蹤器轉變為複雜的 RPG 風格自我提升系統。

### ✨ 新增功能

#### 核心系統
- 🎯 **五大屬性系統**
  - STR (體力) - 運動鍛鍊追蹤
  - INT (智力) - 學習能力提升
  - MP (魔力/靈性) - 靈性探索記錄
  - CRT (創造力) - 藝術創作追蹤
  - GOLD (財力) - 事業與收入管理

- 📊 **雷達圖視覺化** - 即時顯示五大維度發展狀況
- ❤️ **HP 能量條** - 綜合生活作息與健康評分
- 📈 **階梯式獎勵系統** - OK / Great / Best 三級成就

#### 任務系統
- 💧 **智能飲水追蹤** - 未達 2400cc 自動置頂警告
- 🏃 **運動追蹤** - 慢跑、重訓、HIIT 三項記錄
- 📚 **學習追蹤** - 閱讀、語言、課程三維學習
- 🙏 **靈性探索** - 讀經、禱告、日記記錄
- 🎨 **創意實踐** - 練琴、畫畫時間追蹤
- 💰 **財富管理** - 收入記錄與事業行動追蹤

#### 特殊功能
- 💗 **共鳴系統** - 慶祝按鈕帶有愛心飄浮動畫特效
- 📝 **感恩筆記** - 記錄每日感恩時刻
- 🍺 **酒精審計** - 非遊戲化的理性記錄區域，無獎勵機制

#### Google Sheets 整合
- ☁️ **雲端同步** - 所有數據自動同步到個人 Google Sheet
- 🔐 **數據自主** - 數據儲存在用戶自己的 Google 帳號中
- 📊 **靈活分析** - 可使用 Google Sheets 強大功能進行分析
- 🛠️ **自動初始化** - 應用自動創建所需的 Sheet 結構

### 🎨 UI/UX 改進
- 🌑 **深色 RPG 風格** - 完整的遊戲化介面設計
- 💜 **紫色主題** - 統一的視覺語言
- ✨ **動畫特效** - 愛心飄浮、進度條動畫
- 📱 **完全響應式** - 手機和桌面完美適配
- 🎯 **任務優先級** - 飲水不足自動置頂提醒

### 🔧 技術更新
- ➕ 新增 `recharts` - 雷達圖視覺化
- ➕ 新增 `date-fns` - 日期處理
- 🔄 升級 Tailwind CSS 到 v4
- 📦 新增 Google Sheets API 整合服務
- 🏗️ 完全重構組件架構

### 📊 數據管理
- 🕐 **每日自動重置** - 凌晨 4:00 自動重置任務狀態
- 💾 **雙重持久化** - localStorage + Google Sheets
- 📈 **長期追蹤** - 累計天數記錄

### 📚 文檔
- ✅ 完整的 Google Sheets 設置指南
- ✅ 更新所有現有文檔以反映新功能
- ✅ 新增詳細的專案結構說明

### 🗑️ 移除
- ❌ 移除舊的 100 天網格追蹤系統
- ❌ 移除簡化版的統計卡片
